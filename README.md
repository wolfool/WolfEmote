# WolfEmote

이모트 시스템 for Minecraft Paper 1.21+

## Features
- 🎭 10가지 기본 이모트 (앉기/춤/인사/절/울기/웃기 등)
- 💺 앉기/눕기 (아머스탠드 기반, 움직이면 자동 해제)
- ✨ 이모트별 고유 파티클 애니메이션
- 🔊 이모트별 사운드 효과
- 📋 GUI 선택 메뉴
- ⏰ 쿨타임 시스템
- 📢 주변 30블록 플레이어에게 이모트 메시지

## Emotes
| 이모트 | 명령어 | 설명 |
|---|---|---|
| 💺 앉기 | `/emote sit` | 바닥에 앉음 |
| 👋 인사 | `/emote wave` | 손 흔들기 |
| 💃 춤추기 | `/emote dance` | 파티클 춤 |
| 🙇 인사(절) | `/emote bow` | 공손한 인사 |
| 😢 울기 | `/emote cry` | 눈물 파티클 |
| 😄 웃기 | `/emote laugh` | 행복 파티클 |
| 👉 가리키기 | `/emote point` | 방향 가리킴 |
| 😴 잠자기 | `/emote sleep` | 바닥에 누움 |
| 👏 박수 | `/emote clap` | 불꽃 파티클 |
| 🤦 한심 | `/emote facepalm` | 연기 파티클 |

## Commands
| 명령어 | 설명 |
|---|---|
| `/emote` | GUI 열기 |
| `/emote <이름>` | 이모트 바로 실행 |
| `/emote reload` | 설정 리로드 |

## Permissions
| 퍼미션 | 설명 |
|---|---|
| `wolfemote.use` | 이모트 사용 (기본: true) |
| `wolfemote.admin` | 리로드 권한 |
